# Sarv Web

Built with Vue 3, Vite, and Tailwind Css.

## Instalation

Clone Repository.

```bash
git clone git@github.com:ibrahimalanshor/sarv-web.git
```

Install Dependencies.

```bash
npm install
```

Copy Environment.

```bash
cp .env.example .env
```

## Dev Server

```bash
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
```
